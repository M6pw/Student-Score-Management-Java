# Student Score Management System 🎓

Student Score Management System is a Java application that manages student information and course scores. The program uses Object-Oriented Programming concepts to create a Student class that stores student details and handles multiple course scores.

## Project Description
This application allows users to enter a student's ID, name, and number of courses. It then stores the scores for each course, retrieves a specific score based on the course index, and displays all student information with their scores.

The project also demonstrates the use of exception handling to manage invalid inputs and prevent program errors during execution.

## Features
- Create and manage student records.
- Store multiple course scores using arrays.
- Retrieve a specific course score by index.
- Display complete student information.
- Handle incorrect user input and invalid indexes.

## Technologies Used
- Java
- Object-Oriented Programming (Classes & Objects)
- Arrays
- Exception Handling
- Scanner for user input

## Learning Objectives
This project demonstrates fundamental Java concepts, including class design, constructors, arrays, methods, user interaction, and handling runtime errors using exceptions.
